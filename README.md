下载48官方平台历史视频文件的ts流。   
start.py为项目入口。  
**python版本为py3版本。**

---

### 使用模块：
* pyinstaller：打包
* pyqt5：ui
* pyquery：数据处理
* lxml：pyquery的依赖

---

### 视频处理
视频处理使用了ffmpeg，请自行下载并放入目录中。
